# File Descriptions

## CircularBuffer.java: An implementation of a FIFO circular buffer

## Producer.java: Adds a random number to the CircularBuffer at the tail variable location

## Consumer.java: Consumes the next number in the buffer pointed to by head variable

## ProducerConsumer.java: The driver program. Creates separate threads for Consumer.java and
Producer.java and starts their execution.

## Makefile: A Makefile made using information provided in the linked Infospaces
video ending in 283.
